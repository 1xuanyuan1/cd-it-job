# cd-it-job
针对成都IT小伙伴们的工作机会（招聘，求职）交流的微信订阅号项目。

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
